# pack-unpack

O script compress.sh compacta arquivos usando uma das várias ferramentas de compactação, incluindo tar, gzip, bzip2, zip e rar. O usuário é solicitado a escolher a ferramenta de compactação e o script cria um arquivo de saída compactado com a ferramenta e a extensão de arquivo escolhidas. Uma barra de progresso é exibida enquanto a compactação está em execução e uma mensagem de conclusão bem-sucedida é exibida no final.

## Pré-requisitos

- shell bash
- Ferramentas de compressão (tar, zip, rar)

## Instalação

Para usar o script compress.sh, siga estas etapas:

1 - Clone o repositório: `git clone https://github.com/rib-thiago/pack-unpack.git`
2 - Navegue até o diretório onde o script está localizado: `cd compress`
3 - Torne o script executável: `chmod +x compress.sh`
4 - Execute o script: `./compress.sh`
